API Reference

Indonesia Regions API

Authenticated endpoints for Indonesian administrative region data, generated from versioned JSON and guarded by Supabase login.

OpenAPI JSON Markdown
Base URL /
User Auth Bearer JWT
Admin Auth x-admin-token
Static JSON 403 blocked

Auth Flow

  1. Create or invite users in Supabase Auth.
  2. Call POST /api/auth/login with email and password.
  3. Send Authorization: Bearer <access_token> on region data requests.
